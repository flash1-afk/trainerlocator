Leaflet Map Integration Explained - Line-by-Line Guide for Beginners

This document explains exactly how the map functionality works in your StudentDashboard.jsx file.
We use the "Leaflet" library, which is a popular open-source tool for interactive maps.

================================================================================
Files involved: client/src/components/user/StudentDashboard.jsx

> This variable decides if the map is visible or hidden. 'false' means it's hidden by default.

Line 14:   const [userLocation, setUserLocation] = useState(null);
> This stores the user's GPS coordinates (Latitude and Longitude) once we find them.

Line 15:   const mapRef = useRef(null);

4. INITIALIZING THE MAP (Loading the Library)
================================================================================
Function: initializeMap() (Lines 148-165)

Line 149:    if (!mapRef.current || !userLocation) return;
> Safety check: Don't try to draw a map if the HTML div isn't there or we don't know where the user is yet.

Line 152:    if (!window.L) {
> Checks "Is Leaflet loaded?". 'L' is the global name for the Leaflet library.
> If it's NOT loaded, we manually add the Leaflet CSS (Line 153) and Leaflet JS script (Line 158) to the page.
> This is a "Dynamic Import". It keeps your website fast by only downloading the map software when the user actually CLICKS "Show Map".

Summary:
1. We ask browser for GPS.
2. User clicks "Show Map".
3. We download Leaflet.js.
4. We draw the map into the div.
5. We calculate coordinates for user + trainers.
6. We place markers.
7. If user clicks a marker popup button, we signal React to open the Booking Modal.
